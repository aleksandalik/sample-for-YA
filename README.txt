Цель задания для прохождения в дальнейший этап Школы Менеджеров Яндекса 2019 
по направлению Продуктовой Аналитики заключается в определении показателей маркетинговой кампании:
Сколько компания готова платить за одного платящего и одного нового пользователя, чтобы продукт окупался.

Решение в большей мере носит характер описательной аналитики, без применения каких-либо алгоритмов 
(поскольку они не требовались).

Файлы:
data.rar, содержащий:
	visits.txt - документ, содержащий логи о посещениях пользователями сервиса по покупке билетов.
	purchases.txt - документ, содержащий информацию о покупках пользователями.
analytics.py - документ с решением и кодом.
